<p>Netdim is a network design and planning software. Created in April 2016, it is still at an early stage of its development.
A network in NetDim is made of:</p>

<ul>
<li>devices (router, optical switch, host machine or antenna)</li>
<li>physical links (<a href="https://en.wikipedia.org/wiki/Link_aggregation">trunks</a>)</li>
<li>routes </li>
<li>traffic links</li>
<li><a href="https://en.wikipedia.org/wiki/Autonomous_system_%28Internet%29">autonomous systems</a> (AS)</li>
</ul>

<p>A physical link in NetDim is called a trunk: 
it represents a set of physical links aggregated together. 
An autonomous system is a set of devices exchanging routing and signalization messages to carry the incoming traffic. 
The path of a data flow inside an AS depends on the <a href="https://en.wikipedia.org/wiki/Communications_protocol">protocol</a> used
in the AS.</p>

<p><a href="https://github.com/mintoo/networks/blob/master/Readme_ss/introduction.PNG" target="_blank"><img src="https://github.com/mintoo/networks/raw/master/Readme_ss/introduction.PNG" alt="Netdim" style="max-width:100%;"></a></p>
